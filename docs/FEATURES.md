# 🚀 MediCore HMS - Features

## Overview

MediCore HMS is a modular healthcare management platform designed for Patients, Doctors, and Administrators.

The project is being developed incrementally with a focus on scalability, maintainability, security, and future AI-powered healthcare services.

---

# Authentication & Authorization

### Current Features

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Role-Based Access Control

### Supported Roles

* Administrator
